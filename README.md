# 온그리디언츠 패키지·굿즈 컨셉 워크북

온그리디언츠 사례 기반 강의 자료를 초보자용 자기주도 학습 웹사이트로 재구성한 정적 웹앱입니다.
학습자는 어려운 용어를 먼저 이해하지 않아도, 화면에 있는 순서대로 프롬프트를 복사하고 AI 답변을 빈칸에 붙여 넣으며 최종 이미지 생성 프롬프트를 만들 수 있습니다.

## 주요 기능

- 패키지 컨셉, 굿즈 컨셉, 통합 검증을 순서대로 학습
- 단계별 입력값을 브라우저에 자동 저장
- 현재 입력값을 반영한 AI 프롬프트 복사
- 예시 자동 채우기
- 초보자용 용어 풀이와 단계별 "지금 할 일" 안내
- 통합 검증 체크리스트와 진행률 표시
- 최종 기획 문장 자동 생성
- 패키지 디자인 이미지 프롬프트 3개 자동 생성
- 굿즈 이미지 프롬프트 3개 자동 생성
- 전체 내용 또는 개별 이미지 프롬프트 복사, 텍스트 저장

## 실행

정적 파일만으로 동작합니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173/ongredients-self-learning/` 또는 프로젝트 폴더에서 서버를 열면 됩니다.
