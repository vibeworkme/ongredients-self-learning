# 온그리디언츠 패키지·굿즈 컨셉 워크북

온그리디언츠 사례 기반 강의 자료를 자기주도 학습용 웹사이트로 재구성한 정적 웹앱입니다.

## 주요 기능

- 패키지 컨셉, 굿즈 컨셉, 통합 검증을 순서대로 학습
- 단계별 입력값을 브라우저에 자동 저장
- AI 프롬프트 복사
- 통합 검증 체크리스트와 진행률 표시
- 최종 제출 문장 자동 생성, 복사, 텍스트 저장

## 실행

정적 파일만으로 동작합니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173/ongredients-self-learning/` 또는 프로젝트 폴더에서 서버를 열면 됩니다.
